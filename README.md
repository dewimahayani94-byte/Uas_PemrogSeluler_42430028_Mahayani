## 📱 Deskripsi Project
Aplikasi ini adalah katalog digital interaktif untuk lini *smartphone* Samsung Galaxy A Series. Dibangun sebagai pemenuhan Tugas/UAS mata kuliah Pemrograman Seluler, aplikasi ini menerapkan antarmuka responsif, navigasi perpindahan halaman (Intent), serta pengelolaan struktur data secara mandiri menggunakan *Array*. 

Aplikasi ini mendemonstrasikan algoritma murni (tanpa fungsi bawaan instan) berupa **Linear Search** untuk fitur pencarian dan **Bubble Sort** untuk pengurutan data, serta tidak bergantung pada *database* eksternal.

## ⚙️ Fitur Utama
- **UI Responsif (Modul 2 & 3):** Mendukung tata letak *Portrait* dan pembagian layar parsial pada mode *Landscape*.
- **Navigasi Intent (Modul 4 & 5):** Perpindahan data gambar dan teks dari halaman utama ke halaman detail spesifikasi.
- **Pencarian Data (Modul 6):** Implementasi algoritma *Linear Search* untuk memfilter katalog berdasarkan *input* teks.
- **Pengurutan Data (Modul 7):** Implementasi algoritma *Bubble Sort* untuk mengurutkan katalog secara Ascending (A-Z) dan Descending (Z-A).
- **Error Handling & Logcat (Modul 9):** Penerapan `try-catch` untuk mencegah *force close* dan pelacakan aktivitas di latar belakang menggunakan NIM developer.
